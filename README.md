# Garmin vivoactive Binary Watch Face

The Garmin vivoactive allows the inclusion of custom code.  In this case, it is a custom Watch Face.  The goal is a Watch Face that displays the time in binary and contains the other nicities (date, battery, seconds) included in the default Watch Face.
